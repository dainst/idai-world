• The project configuration can be customized and extended in the application itself. In the new configuration editor, you can now make all the changes that previously required manually created and edited JSON configuration files.
  • The project configuration is automatically shared with other users of the same project via an existing synchronization connection, ensuring that all project participants work with the current configuration.
  • A synchronization connection can now optionally be used to transfer also original images instead of just the smaller preview images.
The application download and a full list of changes can be found on Github.
