Field Connect Release

The new QGIS plugin \"Field Connect\" is now available for download and installation from the official QGIS plugin repository. Field Connect was developed by Oliver Zapiec (VZG) as part of the collaboration between the German Archaeological Institute and the Head Office of the Common Library Network (VZG), and provides a simple way of exchanging data between QGIS and Field Desktop in both directions. The plugin provides valuelists as well as field labels and descriptions from Field projects in QGIS and saves the data imported from Field Desktop either in the GeoPackage format or as temporary QGIS layers. The user interface of the plugin is currently available in German and English. If you'd like to contribute translations into other languages, please feel free to contact us. As with Field Desktop, the translation process takes place via the Weblate platform.
